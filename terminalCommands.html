COMMAND LINE COMMANDS

  • pwd : location in the directory
  • cd name-of-folder – access a directory
  • cd ~/name-of-folder
  • mv: to rename a file
  • ls – check the files inside the directory
  • ls –la : list all the files in the directory
  • touch – create a file
  • crtl a – beginning of the line
  • ctrl c – will not execute the command
  • rm –rf name-of-the-file
  • explorer . : to open the explorer in windows
  • clear – clear the screen
