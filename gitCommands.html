GIT COMMANDS

  • GIT clone url
  • GIT status
  • GIT log (to check the commit is there)
  • GIT add .
  • GIT add name-of-the-file
  • GIT add –A
  • GIT commit –m “Message” (messages has to be detailed to specify the changes made to the code)
  • GIT pull origin master: – Show changes made in the repository by others
  • GIT push origin master : To send the local computer repository to GitHub
  • GIT remote –v (View information about a repository)
  • GIT remote add upstream URL
  • GIT fetch upstream
  • GIT merge upstream/master
  • GIT –version
  • GIT config –list – Provide a list of your configuration values
  • GIT help verb or GIT verb --help
  • GIT Init – to start using GIT to track a repo in github
  • GIT reset name-of-the-file – to remove files from the staging area
  • GIT reset
  • GIT diff – shows the changes made in the code
